# Argus

Repositorio de trabajo para utilidades de mercado y procesamiento de datos.

## Módulos

- `forex-factory/`: herramientas para capturar el calendario de Forex Factory, convertir el snapshot a CSV y consultar eventos en JSON.

## Uso rápido

```bash
./tests/run-forex-factory.sh
```

## Notas

Cada módulo mantiene su propio `README.md` con comandos y parámetros detallados.
